Este proyecto contiene varios ejercicios diseñados para evaluar habilidades en desarrollo backend. A continuación se detallan los ejercicios realizados

-Ejercicio 1: Consumo de APIs

Objetivo
Crear una aplicación Node.js que consuma la API de GitHub y muestre los 10 repositorios más populares del usuario "google".

Requisitos Previos

Node.js (https://nodejs.org/)

npm (https://www.npmjs.com/)

Configuración

Clona este repositorio o descarga los archivos en un directorio.

Abre una terminal en el directorio del proyecto y ejecuta el siguiente comando para instalar las dependencias:

npm install axios

Ejecución

Ejecuta el script con el siguiente comando:

node index.js

La aplicación mostrará en la consola los 10 repositorios más populares del usuario "google" ordenados por el número de estrellas en orden descendente.

-Ejercicio 2: Nomenclatura

Objetivo:

Reemplazar los nombres de las variables con nombres más descriptivos que reflejen mejor su función en el código proporcionado.

Descripción
El código original utilizaba nombres de variables poco descriptivos. Se ha refactorizado para utilizar nombres más claros y significativos, mejorando la legibilidad y mantenibilidad del código. No es necesario ejecutar este ejercicio, simplemente se ha mejorado la claridad del código.


-Ejercicio 3: Pensamiento Lógico

Objetivo:

Escribir una función que tome un número entero como entrada y devuelva un array con todos los números enteros impares desde 1 hasta el número de entrada.

Requisitos Previos

Un navegador web moderno.

Ejecución

En este repositorio Dirigete a la carpeta Pensamiento Lógico

Abre el archivo index.html en un navegador web.

Ingresa un número en el campo de entrada y haz clic en el botón "Show Odd Numbers".

Los números impares desde 1 hasta el número ingresado se mostrarán en la página.
